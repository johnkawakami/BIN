This text editor uses the markdown computer language to convert human-readable text into html.

To edit text, type it into the left pane. A preview appears to the right. The resultant HTML code (which you can paste into a website's input box) is above.

To learn more about markdown read this cheatsheet.

The main advantage of markdown is that, in its simplest form, you cannot easily produce bad html. Erroneous markdown is overlooked, and the resultant html code is still valid.

The advantages of this particular web-based editor are as follows

  1. input is auto-saved every minute
  2. there's a live preview
  3. the "copy html code" button puts the code into your clipboard

Note: this app is designed for firefox and nothing else.
